## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mhmdmrabet/whichmovie
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Launch the Project

- ```sh
  npm run dev
  ```
